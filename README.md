# PokeStats

## PokeStats Web Aplication using Node.JS, Express.JS, EJS and the PokeAPI 
### To run the project: 


- type "npm i" into the terminal to install all the package dependencies 
- type "nodemon .\index.js" to run the project on the localhost:3000
